C Programming

C is a powerful general-purpose programming language. It is fast, portable, and widely used in systems programming, game development, embedded systems, and more.

Basics of C

  • C programs are written in plain text and saved with a .c extension.
  • C uses a compiler to convert the source code into executable code.
  • The basic structure of a C program includes libraries, a main function, and statements.
  • C supports low-level memory manipulation, making it highly efficient for performance-critical applications.

Data Types in C

C provides a variety of data types to handle different kinds of data. Below are the commonly used data types in C:

  • int - Used for storing integer values (e.g., int x = 10;).
  • float - Used for storing floating-point numbers (e.g., float y = 3.14;).
  • char - Used for storing single characters (e.g., char letter = 'A';).
  • double - Used for storing double precision floating-point numbers.
  • void - Represents the absence of type (e.g., in functions that do not return a value).

Functions in C

In C, functions allow you to organize code into reusable blocks. Functions are defined with a return type, name, and parameters.

Function Syntax:

return_type function_name(parameter1, parameter2, ...) {
    // body of the function
}

Example: A simple function to add two numbers:

#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 5);
    printf("Sum: %d\n", result);
    return 0;
}

Basic Input/Output

#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("You are %d years old.\n", age);
    return 0;
}

Simple Conditional Statement

#include <stdio.h>

int main() {
    int number;
    printf("Enter a number: ");
    scanf("%d", &number);

    if (number % 2 == 0) {
        printf("%d is even.\n", number);
    } else {
        printf("%d is odd.\n", number);
    }
    return 0;
}
