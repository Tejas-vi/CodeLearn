Java Programming

Java is a high-level, class-based programming language that is designed to have as few implementation dependencies as possible. It is used in web development, software development, and mobile applications.

Basics of Java

  • Java is an object-oriented language, which means everything in Java is treated as an object.
  • Java programs must be compiled into bytecode and run on the Java Virtual Machine (JVM).
  • The basic structure of a Java program includes a public class and the main method.

Data Types in Java

Java offers several built-in data types for different data manipulations:

  • int - For integers (e.g., int x = 10;).
  • float - For floating-point numbers (e.g., float y = 3.14f;).
  • char - For characters (e.g., char letter = 'A';).
  • boolean - For true or false values (e.g., boolean flag = true;).

Basic Input/Output

import java.util.Scanner;

public class UserInput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter your age: ");
        int age = scanner.nextInt();
        System.out.println("You are " + age + " years old.");
    }
}

Conditional Statements

public class EvenOdd {
    public static void main(String[] args) {
        int number = 10;
        if (number % 2 == 0) {
            System.out.println(number + " is even.");
        } else {
            System.out.println(number + " is odd.");
        }
    }
}
