HTML - HyperText Markup Language

HTML is the standard markup language used to create web pages. It structures the content on the web using elements like headings, paragraphs, images, and links.

Basics of HTML

  • HTML is made up of elements, which are enclosed in angle brackets (e.g., <div>).
  • Every HTML page starts with <html> and ends with </html>.
  • HTML documents are divided into the head and body sections using <head> and <body>.

Common HTML Elements

HTML elements are the building blocks of web pages. Here are some commonly used elements:

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <a> - Links
  • <img> - Images
  • <div> - Division or container for other elements

Code Snippets

Basic HTML Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>Welcome to my first HTML page.</p>
  </body>
</html>

Creating Links

<a href="https://www.example.com">Click here</a>

Embedding an Image

<img src="image.jpg" alt="A description of the image">
